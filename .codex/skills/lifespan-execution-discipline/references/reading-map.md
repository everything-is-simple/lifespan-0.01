# 阅读顺序图

## 全局入口

进入 `H:\lifespan-0.01` 后，不要直接改代码。
默认先读：

1. `docs/README.md`
2. `docs/01-design/00-system-charter-20260409.md`
3. `docs/01-design/01-doc-first-development-governance-20260409.md`
4. `docs/02-spec/00-repo-layout-and-docflow-spec-20260409.md`
5. `docs/02-spec/01-doc-first-task-gating-spec-20260409.md`
6. `docs/03-execution/README.md`

## 执行闭环入口

继续正式执行时，默认按下面顺序进入：

1. `docs/03-execution/00-conclusion-catalog-20260409.md`
2. `docs/03-execution/B-card-catalog-20260409.md`
3. `docs/03-execution/C-system-completion-ledger-20260409.md`

## 继续现有执行线时

如果是接着做一张已有执行卡，顺序是：

1. 先读当前 `conclusion`
2. 再读当前 `card`
3. 再读当前 `evidence`
4. 最后读当前 `record`

不要只靠一张历史卡片重开旧线；如果后面已有更新结论，以更新结论为准。
